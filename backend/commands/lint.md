# Comandos de Linting e Formatação

Para manter o código limpo e padronizado, utilizamos ESLint e Prettier.

### Verificar erros de Lint
```bash
npm run lint
```

### Corrigir erros de Lint automaticamente
```bash
npm run lint:fix
```

### Formatar todo o código
```bash
npm run format
```

---
**Dica:** Se você usa VS Code, instale as extensões **ESLint** e **Prettier - Code formatter** para ver os erros em tempo real e formatar ao salvar.
